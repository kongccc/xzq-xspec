# 任务执行AI Agent

专门用于复杂任务分解和执行的Agent模板，支持多步骤任务规划和执行监控。

## 核心能力

- **任务分解**: 将复杂任务分解为可执行的子任务
- **规划生成**: 自动生成任务执行计划
- **执行监控**: 实时监控任务执行状态
- **错误恢复**: 任务失败时的自动恢复机制

## 任务执行流程

```
任务接收 → 任务分析 → 计划生成 → 执行监控 → 结果汇总 → 任务完成
```

## 任务类型支持

### 1. 信息处理任务
- 数据收集和整理
- 信息分析和汇总
- 报告生成

### 2. 系统操作任务
- 文件操作
- 系统配置
- 自动化脚本执行

### 3. 创作任务
- 内容创作
- 代码编写
- 设计生成

## 快速开始

```javascript
// 创建任务Agent
const taskAgent = new TaskAgent({
  maxSteps: 20,        // 最大执行步骤
  timeout: 300000,     // 超时时间（5分钟）
  retryCount: 3        // 重试次数
});

// 执行复杂任务
const result = await taskAgent.executeTask({
  goal: "收集最近一周的AI新闻并生成摘要报告",
  constraints: ["使用中文", "包含至少5条新闻"],
  tools: [webSearchTool, summaryTool]
});
```
